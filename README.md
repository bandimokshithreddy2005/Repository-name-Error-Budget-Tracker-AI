# 📌 Overview

**Error Budget Tracker with AI** is an intelligent **Streamlit-based Site Reliability Engineering (SRE) Dashboard** designed to monitor service reliability and analyze error budgets using Artificial Intelligence.

The dashboard enables engineering teams to:

* ✅ Monitor Service Health
* 🎯 Calculate Error Budgets
* 📊 Visualize Reliability Metrics
* 🔥 Analyze Burn Rate
* 🤖 Generate AI-powered SRE Reports using Google Gemini AI
* ⚡ Gain Real-Time Reliability Insights

# 🚀 Installation

```bash
# Clone the Repository
git clone https://github.com/YOUR_USERNAME/Error-Budget-Tracker-AI.git

# Move into Project Directory
cd Error-Budget-Tracker-AI

# Install Dependencies
pip install -r requirements.txt

# Run the Application
streamlit run streamlit_app.py
```
